=== Citadela Pro ===
Requires at least: 5.8
Tested up to: 6.0
Requires PHP: 7.4.0

Ready to use layouts and blocks for Citadela with many customization options

== Description ==
Citadela PRO plugin extends free Citadela theme with advanced options that will allow you to customize look & feel of your website. Using WordPress Customizer you will be able to change fonts, configure color, choose different header, footer, sidebars and other layout settings.
