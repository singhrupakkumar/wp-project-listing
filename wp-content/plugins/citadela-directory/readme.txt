=== Citadela Listing ===
Requires at least: 5.8
Tested up to: 6.1
Requires PHP: 7.4.0

Universal listing features for any website

== Description ==
= Spice up your Business Website or Blog with Listing Features =

Thanks to the Citadela Listing plugin, you can add universal listing functionality to Citadela or any standard WordPress theme. You can create a Directory portal, Portfolio website, Job Offers, Classified website, Recipes website, Travel guide and much more. Possibilities are endless. Activate the plugin and start using all unique listing blocks.
